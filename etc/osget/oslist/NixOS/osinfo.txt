NixOS is a Linux distribution built on top of the Nix package manager. It uses declarative
configuration and allows reliable system upgrades. Two main branches are offered:
current Stable release and Unstable following latest development. Although NixOS
started as a research project, it is a fully functional and usable operating system.
NixOS has tools dedicated to DevOps and deployment tasks.
